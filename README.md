# Event Talks App

This repository contains a simple, single-page web application for a 1-day technical talk event.

## Features:
- Displays a schedule of talks with titles, speakers, categories, and descriptions.
- Includes a 1-hour lunch break.
- Allows searching talks by category (client-side).

## Technologies:
- HTML
- CSS
- JavaScript

## How to Run Locally:

1.  Save the `index.html`, `style.css`, and `script.js` files to a directory on your computer.
2.  Open your terminal or command prompt, navigate to that directory.
3.  Run a simple Python HTTP server (if you have Python installed):
    ```bash
    python3 -m http.server 8000
    ```
4.  Open your web browser and go to `http://localhost:8000`.
